§1

What is Analogy?

▾

Analogy tests your ability to identify the relationship in a given pair and apply the same relationship to a new pair.

  • Format: A : B :: C : ? — Find what completes the pair with the same logic.
  • Types tested: Word, Number, Letter, Mixed, Pair-based, Coding-based.
  • Weightage: 3–5 questions per SSC CGL Tier-1 paper (15–20 marks effectively with accuracy).
  • Key skill: Pattern recognition within 10 seconds — the faster you find the relation, the more you score.
  • SSC twist: Questions always have 2–3 options that partially match; final answer requires complete logic match.
⚡ Golden Rule: Never assume — always verify both pairs with the SAME operation before marking the answer.

Key Observations from PYQs

  • Number Analogy: Squares (n²) appear in 60% of papers. Learn squares of 1–50.
  • Word Analogy: Worker-Workplace, Tool-Work, Country-Capital appear in every exam.
  • Letter Analogy: Alphabetical position (+2, +3, +n) patterns are most common.
  • Trap: SSC often gives options with correct single pair but wrong logic — check BOTH pairs.
  • Difficult questions use composite operations: (n² + n), (n² - 1), (n³ - n).

Shortcuts & Speed Tricks

▾

Number Shortcut Patterns

Pattern 1: Square Relation
n : n² → If first number small (1–25), always check square first.
Pattern 2: n² ± n
n(n+1) or n(n-1) → 3:12 = 3×4, 5:30 = 5×6
Pattern 3: Difference trick
If diff is same both sides → simple addition/subtraction
Pattern 4: Ratio trick
If A÷B = C÷D (cross multiply) → ratio analogy
Pattern 5: Reverse digits
23:32 → always check reverse for 2-digit numbers

Letter Shortcut Patterns

Trick 1: Position map
A=1 B=2 ... M=13 N=14 ... Z=26. Memorize M=13, N=14 as midpoint.
Trick 2: Opposite pairs
A+Z=27, B+Y=27... If sum=27, it's mirror analogy
Trick 3: Letter gap
Count gap between letters. If consistent, same gap applies.
Trick 4: Vowel positions
Vowels: A=1, E=5, I=9, O=15, U=21. Common in alternating.
Trick 5: String reversal
If the letter group is reversed, apply reversal to second pair too.

Elimination Tricks (Save 30+ Seconds)

  • Eliminate by category: If question is Worker:Workplace, eliminate options that are not workplaces.
  • Eliminate by gender: If pair is Male:Female, all options must follow same gender logic.
  • Eliminate by number range: If pattern gives n² and n=7, answer must be near 49. Eliminate 23, 56, etc.
  • Eliminate by letter position: If letter must be +3 from G (=J), eliminate options starting with other letters.
  • Eliminate by GK: You know India's capital is New Delhi — eliminate wrong capitals from options.
  • 2-wrong rule: If an option has 2 obvious wrong facts, eliminate immediately.
  • One-operation check: Apply the simplest operation first. If it gives an option, verify before jumping to next.

Common Mistakes & SSC Traps

▾

Top 10 Mistakes Students Make

  1. Checking only one pair: Students verify the given pair's logic but forget to apply the SAME logic to the second pair before marking.
  2. Confusing partial match: Option gives 3/4 correct attributes but one wrong. SSC specifically designs such distractors.
  3. Stopping at first working operation: If +3 works for pair 1, check if +3 also works for pair 2. Never assume.
  4. Letter counting errors: Students miscount alphabet positions under time pressure. Practice: K=11, M=13, Q=17, W=23.
  5. Reversing the relation: If Doctor:Hospital is correct, Hospital:Doctor is NOT necessarily valid for another pair.
  6. Wrong GK recall: Students confuse similar capitals (e.g., Colombo vs Sri Jayawardenepura Kotte). Always double-check.
  7. Missing the "odd man out" inside analogy: Some traps have the right answer as option C when all others seem plausible.
  8. Not knowing male-female pairs: Common error area — Ram:Ewe (not Goat), Gander:Goose (not Duck).
  9. Assuming additive when it's multiplicative: 3:9 seems like +6, but it's ×3. Always check multiplication before addition if numbers grow rapidly.
  10. Skipping coding-based questions: These have a fixed method — most students skip them but they're actually solvable with A=1 method.

SSC Trap Designs

🚨 Trap 1: 4:8 :: 9:? → Options: 13, 18, 27, 81. Students pick 13 (×2 fails for 9). Answer: 27 (n² → 2²:2³ :: 3²:3³)
🚨 Trap 2: Letter pair BEH:? :: Three letters with gap of 2 (B,E,H=+3 each) but options show +2 variants.
🚨 Trap 3: Word analogy with two valid-looking relations. Example: Pen:Writer. Options include "Brush:Painter" AND "Keyboard:Typist". BOTH seem right — check which one better mirrors the EXACT relation.

How SSC Designs Distractors

  • Same category, different relation
  • Same relation, different category
  • Correct answer reversed
  • One attribute off (almost correct)
  • Close synonym to correct answer
  • Second pair with slightly wrong number (n²+1 vs n²)
  • Gender swap trap
  • Capital of country vs currency trick

Exam Strategy

▾

How Toppers Solve Analogy

  1. Read pair → Identify category (number/letter/word) in 2 seconds
  2. For numbers: check n² in 3 seconds → found? Verify pair 2 → done.
  3. For letters: count position shift → apply to pair 2 → match with options.
  4. For words: form a sentence → check all options → eliminate 2 → pick remaining.
  5. Stuck after 20 sec? → Use elimination → mark best guess → MOVE ON.

Target Times Per Question

TypeTarget
Easy number/word8–12 sec ✅
Medium letter/GK15–20 sec ✅
Hard/Tricky30 sec max → else skip
Skip & returnIf stuck > 30 sec
Overall analogy (5Q)< 90 seconds total

When to Guess & When to Skip

Guess when: You can eliminate 2 options. 50% chance × 2 marks = +EV with −0.5 penalty. If elimination leaves 2 options, ALWAYS guess.
Skip when: All 4 options look equally valid, you have zero idea of the GK fact needed, AND you've spent 30+ seconds. Mark and return at end.

SSC CGL Marking:
Correct: +2
